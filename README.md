# WordPress Portfolio Website

This is a beginner WordPress project created to understand core concepts of
WordPress such as themes, plugins, pages, posts, menus, and contact forms.

## Pages Included
- Home
- About
- Contact

## Features
- Custom pages and navigation menu
- Blog posts
- Contact form using Contact Form 7
- SEO using Yoast SEO
- Caching using WP Super Cache

## Tools & Technologies
- WordPress
- Astra Theme
- Contact Form 7
- Yoast SEO
- WP Super Cache

## Purpose
This project was created as part of my learning journey to understand
WordPress development fundamentals.
